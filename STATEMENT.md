Project Statement File
1. Problem Statement

Many people struggle to understand how many calories they need daily and what kind of meals they should follow according to their fitness goals. Since calculating BMR, TDEE, and planning meals manually can be confusing, this project aims to create a simple Python program that helps users get a basic diet plan instantly based on their personal details.

2. Scope of the Project

This project provides a basic calorie calculation system and a simple meal suggestion model. It does not focus on medical-level accuracy but offers an easy and beginner-friendly tool for estimating daily calorie requirements and creating a simple meal plan. The project stays within the limits of Class 12 Python concepts like functions, conditionals, and input handling.

3. Target Users

Students and beginners who want a simple diet-planning tool

Individuals looking for a quick idea of their daily calorie needs

People who want a basic vegetarian or non-vegetarian meal plan

Anyone who needs a lightweight diet planner without advanced complexity

4. High-Level Features

Collects basic user information (age, gender, height, weight, activity level, goal)

Calculates BMR using a simple formula

Estimates daily calorie requirement (TDEE)

Adjusts calories based on user goals (shred, bulk, maintain)

Provides a simple veg or non-veg meal plan

Gives daily water intake recommendation

User-friendly and works entirely through text input
