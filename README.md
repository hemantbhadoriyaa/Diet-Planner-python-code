Diet Planner Using Python

A simple Python program that calculates daily calorie requirements and generates a basic meal plan based on user inputs such as age, gender, weight, height, activity level, goal, and diet preference.
________________________________________
 Features
•	Calculates BMR (Basal Metabolic Rate)
•	Calculates TDEE (Total Daily Energy Expenditure)
•	Adjusts calorie needs based on goal:
o	Shred
o	Bulk
o	Maintain
•	Provides a vegetarian or non-vegetarian meal plan
•	Recommends daily water intake
•	Written in simple Python 
________________________________________
 Requirements
•	Python 3
•	No external libraries
•	Works in any terminal or IDE
________________________________________
 How to Run the Program
1.	Clone or download this repository
2.	Open the .py file
3.	Run the script using:
python diet_planner.py
4.	Enter the required details
5.	View your personalized diet plan
